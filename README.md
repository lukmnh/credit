CREDIT SIMULATOR
--

A simple Java-based application to simulate vehicle loan calculations that supports multi-sheet simulation, file input, and CLI execution.


Requirements
-> make sure your environment has Java 21

How to Run Application
--
1. Clone repository : git clone https://github.com/lukmnh/credit.git
   lalu buka IDE (VSCODE / IntelliJ) dan cd credit-simulator
2. Run Application without file
-> ./credit_simulator
3. Run Application with file
-> ./credit_simulator file_inputs.txt
